# 역사 기반 능력치 산정 방법론

> 원작 KOEI 수치를 복제하지 않고, **역사적 근거(정사 삼국지·후한서·연의)에서 재현 가능하게**
> 능력치를 산정하는 규칙. 같은 근거 → 같은 값(재현성). 각 인물의 산정 근거는
> `packages/engine/data/officers.json`의 항목과 본 문서의 룰로 추적된다.

## 원칙
- **근거 기반**: 실제 역할·업적으로 점수를 부여한다. 원작 수치 참조 금지.
- **재현성**: 룰이 명시적이라 제3자가 같은 근거로 유사값을 재산출할 수 있다.
- **연의 보정 최소화**: 사실(정사)을 기본으로, 대중적 이미지(연의)는 매력/무력에 소폭 반영.
- 범위 1..100. 대부분 인물은 40~99대에 분포(1~30은 무능/악명).

## 지력 (智力) — 지모·행정·계략
기준 밴드:
| 밴드 | 값 | 근거 예시 |
|------|----|----------|
| 초월적 군사(軍師) | 95~100 | 국가 전략 설계·연전연승 지략 (제갈량·주유·순욱·곽가급) |
| 일류 책사·명행정가 | 85~94 | 전략 참모·유능한 재상 (사마의·노숙·정욱) |
| 지장(智將)·유능 관리 | 70~84 | 계책 구사 가능한 장수·행정가 |
| 평범 | 50~69 | 일반 무장·관리 |
| 지력 약함 | 30~49 | 용맹하나 지모 부족(장비·허저 유형) |
| 무모·우둔 | 1~29 | 판단 착오로 패망(여포의 전략판단 등) |

가산: 저술·발명·법제 정비(+), 대표적 계략 성공(+). 감산: 치명적 오판·피살(−).

## 무력 (武力) — 개인 무예·전장 용맹
| 밴드 | 값 | 근거 예시 |
|------|----|----------|
| 천하무쌍 | 95~100 | 일기토 최상위 (여포·관우·장비·조운·마초) |
| 맹장 | 85~94 | 일급 전투 지휘·무용 (황충·전위·허저·태사자) |
| 유능 무장 | 70~84 | 실전 지휘 가능 |
| 평범 | 50~69 | 일반 |
| 문관형 | 30~49 | 무예 약함(책사·행정가) |
| 비전투 | 1~29 | 순수 문관·군주 참모 |

가산: 유명 일기토 승리·단독 전공(+). 연의 각색 무용은 최대 +5 이내.

## 매력 (魅力) — 인망·통솔·외교·등용력
| 밴드 | 값 | 근거 예시 |
|------|----|----------|
| 천하의 인망 | 92~100 | 군웅의 구심점 (유비·조조 등 창업 군주) |
| 높은 인망·명망가 | 82~91 | 명문·덕망·대군 통솔 (손책·원소·관우) |
| 유능 | 65~81 | 부대·인재를 잘 이끎 |
| 평범 | 45~64 | 일반 |
| 낮음 | 20~44 | 포악·불신 (동탁·여포) |

가산: 세력 창업·인재 집결·백성 지지(+). 감산: 폭정·배신 이력(−).

## 군주 vs 장수
- **창업 군주**는 매력이 핵심(세력 구심점). 조조·유비는 매력 최상위.
- 무력이 낮아도 지력·매력으로 군주 역할 수행 가능(유비 무력 68이지만 매력 99).

## 도시 초기 수치 산정 (시나리오)
역사적 규모로 밴드화(밸런싱 전 기준선):
| 지표 | 대도시(낙양·업·성도) | 중도시 | 소도시/변경 |
|------|---------------------|--------|-------------|
| 인구 | 350k~450k | 200k~330k | 80k~180k |
| 병사 | 10k~15k | 5k~9k | 2k~4k |
| 금 | 600~900 | 400~600 | 150~400 |
| 군량(쌀) | 900~1400 | 600~900 | 300~600 |
| 토지/치수 | 550~650 | 480~550 | 400~480 |
| 민심 | 세력 안정도·통치 성향 반영 45~70 |

폭정 세력(동탁)은 민심 하향, 덕치 세력(유비)은 민심 상향 보정.

## 데이터 반영 규칙
- 각 officer는 위 밴드에서 근거로 값을 정한다. 이상치는 리뷰(data-quality).
- 값 변경 시 근거를 커밋 메시지/PR에 남긴다(감사 추적).
- **밸런싱 튜닝**은 산정 후 별도 단계 — 게임성 위해 조정하되 역사 밴드를 크게 벗어나지 않는다.
