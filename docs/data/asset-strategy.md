# 이미지·에셋 전략

> 원작 삼국지2의 **초상화·지도 타일·UI 스프라이트·음악은 KOEI 저작물**이라 재사용 불가.
> 웹 리뉴얼은 자체/오픈 라이선스 에셋으로 구성한다. 여기서는 필요한 에셋 목록과 방침을 정의.

## 필요한 에셋 카탈로그

| 에셋          | 용도             | 규모             | 방침                                                            |
| ------------- | ---------------- | ---------------- | --------------------------------------------------------------- |
| 장수 초상화   | 인물 카드·대화   | 장수 수백        | 오리지널 일러스트 / AI 생성 / 오픈아트. **원작 초상 복제 금지** |
| 지도 (41주)   | 메인 전략 화면   | 1 대형 + 41 노드 | 자체 제작 벡터 지도(역사 지리 기반)                             |
| UI 스킨       | 프레임·버튼·폰트 | 세트             | 오리지널 디자인. 오픈 폰트(예: Noto Sans KR)                    |
| 아이콘        | 금·쌀·병사·명령  | 수십             | 오픈 아이콘셋(라이선스 확인) 또는 자체                          |
| 이펙트/사운드 | 전투·일기토·BGM  | 세트             | 오픈/자작. **원작 음원 사용 금지**                              |

## 라이선스 원칙 (ai-env security/governance 반영)

- 모든 에셋은 **출처·라이선스를 기록**한다(`apps/web/public/assets/CREDITS.md`).
- 상업/공개 배포 가능 라이선스만 사용(CC0/CC-BY/오픈폰트/자작). 불명확하면 사용 안 함.
- AI 생성 이미지는 사용한 모델·프롬프트·라이선스 조건을 기록.

## 렌더링 방향 (채택: 절차적)

- **채택 = 데이터 기반 절차적 렌더링**(SVG). 원작 저작물 없이 구현·재현 가능.
- **초상**: `apps/web/src/Portrait.tsx` — 장수의 정적 속성(이름 해시·능력치·연령·역할·진영)에서
  결정론적으로 얼굴을 생성. 군주=관(冠), 무장(무력≥85)=투구, 문관(지력≥82)=문사건, 그 외 두건.
  연령으로 수염 길이·백발, 무력으로 눈썹 각도. 같은 장수 → 항상 같은 그림.
- **지도**: 41개 주 노드-그래프 SVG(인접선 = 침공 경로).

## 향후 (선택)

- 절차적 렌더로 게임이 완결돼 있으므로, 원하면 실제 일러스트로 **점진 교체** 가능
  (오리지널/오픈 라이선스). 교체 시에도 데이터·컴포넌트 계약은 그대로.
- 아트 톤(도트 리메이크/모던 플랫/수묵)은 실제 아트 도입 시 결정.
